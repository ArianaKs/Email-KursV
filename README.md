KursV-master - данный сервис избавляет пользователей от необходимости постоянно вручную проверять курсы валют на сайтах банков или в приложениях. Сервис решает задачу централизованного хранения и управления актуальными курсами валют.

EmailSenderAPI-main - обеспечивает надежную отправку электронных писем с HTML-контентом через SMTP-сервер.

Email-KursV - платформа представляет собой систему из двух независимых веб-приложений для отправки курсов валют по email. Первое приложение (KursV) на порту 7092 хранит и управляет курсами валют, предоставляя их список в формате JSON. Второе приложение (EmailSenderAPI) на порту 7200 отправляет письма и содержит веб-интерфейс для пользователей.
